# Gideon local viseme assets

The `orbit-viseme-v1` and `nova-viseme-v1` packs are project-owned fictional presenter derivatives generated from Gideon's existing approved Orbit and Nova catalog assets. They are used only for the disclosed `AI-generated brand presenter` feature.

Each pack is a fixed 3x3 identity-aligned sprite sheet plus mechanically cropped runtime frames. The packs contain no real-person upload, reference voice, third-party stock artwork, or model weights. Runtime animation is local and does not call the image-generation service that created these release assets.

The pack manifest records the sheet and individual frame hashes, anchor, crop-safe region, supported expressions, chroma key, mouth-frame mapping, provenance, approval, and licence file. Runtime preflight verifies these files and hashes before a pack can render.
